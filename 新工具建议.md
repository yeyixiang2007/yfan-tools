# yfan-tools 可添加的新工具建议

## 📊 数据处理类（补充现有缺失）

### 1. Excel/CSV 处理工具
- **功能**：Excel 文件导入/导出、数据清洗、筛选、排序
- **价值**：数据处理是高频需求
- **技术栈**：`xlsx` 或 `exceljs` 库

### 2. 数据可视化工具
- **功能**：上传 CSV/JSON，生成图表（柱状图/折线图/饼图）
- **价值**：所见即所得的数据分析
- **技术栈**：`chart.js` 或 `echarts`

### 3. JSON Schema 验证器
- **功能**：上传 JSON + Schema，验证数据格式
- **价值**：API 开发和数据处理常用

## 🔧 开发工具类（补充现有缺失）

### 4. API 请求工具
- **功能**：支持 GET/POST/PUT/DELETE，请求头设置，响应查看
- **价值**：调试 API 极其方便
- **技术栈**：`axios` 或 `node-fetch`

### 5. Dockerfile 生成器
- **功能**：选择语言/框架，生成优化的 Dockerfile
- **价值**：容器化部署必备
- **技术栈**：模板引擎 + Dockerfile 模板

### 6. JWT 生成器（增强版）
- **功能**：添加 Header Payload Secret 配置，生成并验证
- **价值**：比现有解码器更完整

## 📝 文档工具类

### 7. API 文档生成器
- **功能**：输入 Swagger JSON，生成 HTML 文档
- **价值**：API 文档自动生成

### 8. README 生成器
- **功能**：输入项目信息，生成规范的 README.md
- **价值**：项目文档快速创建

## 🎨 设计工具类

### 9. 图标生成器
- **功能**：输入文字，生成 SVG 图标（多尺寸）
- **价值**：快速生成图标资源

### 10. 渐变色生成器
- **功能**：选择起始色/结束色，生成 CSS 渐变代码
- **价值**：UI 设计常用

### 11. SVG 转 PNG
- **功能**：上传 SVG，导出为 PNG（多尺寸）
- **价值**：图标/Logo 转换

## 📱 实用工具类

### 12. 二维码解码器
- **功能**：上传二维码图片，解码为文本
- **价值**：二维码扫描替代品

### 13. 图片压缩器
- **功能**：上传图片，调整质量/尺寸，下载压缩后图片
- **价值**：图片优化必备

### 14. 文件 MD5/SHA 计算
- **功能**：上传文件，计算哈希值
- **价值**：文件校验

### 15. 文件大小格式化
- **功能**：输入字节，转换为 KB/MB/GB/自动选择
- **价值**：简单但实用

## 🧪 测试工具类

### 16. 正则表达式可视化
- **功能**：输入正则，高亮显示匹配部分
- **价值**：比现有测试器更直观

### 17. SQL 注入检测
- **功能**：输入 SQL 语句，检测常见注入漏洞
- **价值**：安全测试

## 🌍 网络工具类

### 18. HTTP 请求模拟
- **功能**：模拟 POST/PUT 请求，测试 API
- **价值**：API 测试工具

### 19. 域名 WHOIS 查询
- **功能**：输入域名，查询注册信息
- **价值**：域名查询

### 20. 端口扫描器
- **功能**：扫描目标 IP 常用端口状态
- **价值**：网络诊断

## 📚 学习工具类

### 21. 单词本
- **功能**：添加单词+释义，支持搜索/复习
- **价值**：背单词工具

### 22. 代码片段收藏
- **功能**：添加代码片段，支持分类/搜索/复制
- **价值**：代码管理

### 23. 速查表
- **功能**：常用命令速查（Git/Linux/SQL 等）
- **价值**：快速查询

## 💰 财务工具类

### 24. 利率计算器
- **功能**：复利计算、等额本息还款计算
- **价值**：理财计算

### 25. 货币转换（实时）
- **功能**：输入金额+币种，实时汇率转换
- **价值**：比现有 exchange 更实时

## 推荐优先级

**高优先级（最常用）：**
1. Excel/CSV 处理工具
2. API 请求工具
3. 图片压缩器
4. 二维码解码器

**中优先级（实用）：**
5. Dockerfile 生成器
6. 渐变色生成器
7. 单词本
8. 代码片段收藏

**低优先级（小众）：**
9. API 文档生成器
10. README 生成器
11. 域名 WHOIS 查询
12. 端口扫描器
